Возможности виджета

Кастомизация

  • Цвет темы (primary color)
  • Логотип компании
  • Приветственное сообщение
  • Позиция кнопки (лево/право)
  • Текст на кнопке

Функции

  • Отправка файлов и изображений
  • Quick reply кнопки
  • Индикатор "печатает"
  • Уведомления о новых сообщениях
  • Переход в мессенджеры

Интеграция

  • Один JS-скрипт для установки
  • API для программного управления
  • События (открытие, сообщение)
  • Передача данных пользователя
  • A/B тестирование
