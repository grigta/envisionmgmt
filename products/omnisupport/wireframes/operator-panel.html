Описание элементов

Левая панель (Sidebar)

  • Статус оператора (онлайн/офлайн/перерыв)
  • Навигация: мои диалоги, очередь, все, клиенты
  • Счётчики непрочитанных сообщений

Список диалогов

  • Поиск и фильтрация по статусу
  • Иконка канала (TG/WA/Web)
  • Превью последнего сообщения
  • Теги приоритета

Область чата

  • История сообщений
  • ИИ-подсказки на основе RAG
  • Кнопки быстрых действий
  • Вложения (файлы, изображения)

Карточка клиента

  • Контактные данные из всех каналов
  • Теги и сегментация
  • История всех обращений
  • Внутренние заметки
