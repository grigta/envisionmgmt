# attention.ai

Лендинг для AI-интегратора attention!ai. Статический HTML без сборки.

## Запуск

Открыть `index.html` в браузере. Сервер не требуется.

## Tech Stack

- HTML5 + Tailwind CSS (CDN)
- Google Fonts (Inter)

## Структура

```
├── index.html      # Главная страница
├── privacy.html    # Политика конфиденциальности
└── images/         # Логотипы партнёров (ч/б)
```

## Дизайн-токены

| Token | Value | Usage |
|-------|-------|-------|
| `primary` | #171717 | Основной текст |
| `secondary` | #525252 | Вторичный текст |
| `accent` | #2563EB | Ссылки, кнопки |
| `surface` | #FAFAFA | Фон страницы |
| `surface-elevated` | #FFFFFF | Карточки |
| `border` | #E5E5E5 | Границы |
| `muted` | #737373 | Приглушённый текст |

## CSS-классы анимаций

| Класс | Описание |
|-------|----------|
| `.fade-up` | Scroll-triggered fade-in (+ `.visible`) |
| `.card-lift` | Hover lift для карточек |
| `.btn-shine` | Shine эффект на кнопках |
| `.typing-text` | Эффект печатной машинки |
| `.marquee-container` | Бесконечный горизонтальный скролл |
| `.link-underline` | Анимированное подчёркивание |

## Бренд

- Название: **attention!ai** (без пробела, "ai" в accent цвете)
- Логотип: молния в тёмном скруглённом квадрате
